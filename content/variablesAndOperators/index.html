    Variables and Operators

    متغیر ها

    متغیرها مثل ظرف هایی هستند که حافظه به شما اختصاص میده تا بتونید به اونها مقدار اولیه بدید یا مقدارشون رو تغییر بدید.

    چگونگی استفاده

    در زبان ++C برای ایجاد یک متغیر اول باید نوع متغیر رو مشخص کنید تا کامپایلر متوجه جنس مقداری که قراره درون اون بریزید بشه و بتونه مموری مناسب در اختیار شما قرار بده.

    انوع متغیرهای پر کاربرد رو میتونید در جدول زیر مشاهده کنید.

    Description Data Type
    صحیح یا غلط bool
    عدد صحیح int
    عدد صحیح بزرگ long long
    عدد اعشاری float
    عدد اعشاری دقیق double
    یک کاراکتر char
    یک رشته از کارکترها string

    میتونید لیست کامل تری رو در اینجا ببینید.

    در زبان ++C برای ساخت متغیر بدون مقدار اولیه باید از سینتکس زیر استفاده کنید. 

    <DATA_TYPE> <NAME>;

    یا اگر قراره به اونها مقدار اولیه بدید: 

    <DATA_TYPE> <NAME> = <VALUE>;

    به عنوان مثال. 

    int x;
int y = 2;
long long z = -194870499243;
float f = 1.32;
long long p;
char c = 'A'; //(1)!
string s = "salam!"; //(2)!
bool a = true;
bool b = 0; //(3)!

    1. برای مقدار دهی به کاراکتر باید از single quote استفاده کنید.
    2. برای مقدار دهی به رشته باید از double quote استفاده کنید.
    3. برای مقدار دهی به bool میتونید هم به صورت true یا false و هم به صورت 0 یا 1 مقدار مورد نظر رو تعیین کنید.

    همچنین میشه یک متغیر رو تعریف و در خط های بعدی به اون مقدار داد یا مقدار اون رو عوض کرد.

    int x = 2;
string y;
x = 1;
y = "meghdar!";

    نیازی نیست هر متغیر رو در یک خط تعریف کنید و میشه تمامی متغیر های از یک جنس رو در یک خط تعریف و مقدار دهی کرد. دو کد زیر معادل هستند: 

    int x = 2;
int y;
int z = 3;


    int x = 2, y, z = 3;

// int x = 2, a = "test"; ❌

    همچنین می توان چند متغیر رو یکجا مقدار دهی کرد. 

    int x, y, z;
x = y = z = 2;

    تفاوت int و long long

    از اونجایی که مموری هیچ کامپیوتری نامحدود نیست بنابراین کامپایلر و سیستم عامل، میزان مشخص و معینی حافطه برای هر نوع از متغیر های نام برده شده در نظر می گیرند.

    این مقدار رو بر حسب بایت میتونید با استفاده از تابع sizeof برای هر متغیر ببینید. 

    cout << sizeof(int);        // output: 4
cout << sizeof(long long);  // output: 8
cout << sizeof(bool);       // output: 1
    و این به آن معناست که برای int، \(4 \times 8 = 32\) بیت و برای long long، \(8 \times 8 = 64\) بیت مموری لحاظ می شود.

    • یعنی هر int می تواند عددی بین \(-2^{31} \approx -2.14 \cdot 10^9\) تا \(2^{31} \approx 2.14 \cdot 10^9\) اختیار کند.
    • همچنین هر long long می تواند عددی بین \(-2^{63} \approx -9.22 \cdot 10^{18}\) تا \(2^{63} \approx 9.22 \cdot 10^{18}\) اختیار کند.

    با نگاه بر این اعداد ممکنه استدلال کنید که یک بیت بدون استفاده مونده اما در int، بیت 32 ام و در long long، بیت 64 ام، signed bit نام دارند و برای تعیین علامت عدد استفاده می شوند.

    به طوری که اگر این بیت 1 باشد، عدد منفی و اگر این بیت 0 باشد، عدد مثبت است.

    نام گذاری

    شروط نام گذاری

    برای نام گداری متغیر ها تمامی شروط زیر باید رعایت بشه:

    • نام ها می توانند دارای حروف، اعداد یا underscore (_) باشند.
    • نام ها حتما باید با حروف یا underscore آغاز شوند (نمی تواند نام متغیر با عدد آغاز شود).
    • نام ها به بزرگی و کوچکی حساس هستند، یعنی myVar با myvar متفاوت است.
    • نمی توان از حروف خاص (@, !, %, -, ...) در نامگذاری استفاده کرد.
    • از کلمات کلیدی ++C نمی توان در نامگذاری استفاده کرد.

    Global vs. Local

    متغیر های Global متغیر هایی هستند که داخل اسکوپ (یا همون آکولاد {}) هیچ تابع یا حلقه ای (از جمله تابع main) قرار ندارند.

    int x; // global variable

int f1();
int f2();
// and other functions

int main() {
}

    این متغیر ها توسط تمام توابع یا حلقه های کد شناخته می شوند.

    اما متغیرهای Local تنها داخل همان اسکوپی که در آن تعریف می شوند (و منطقا در خط هایی پایین تر از خط معرف) شناخته می شوند.

    int main() {
    int x; // local variable 

    for (int i = 0 /*(1)!*/; i < x; i++) {
        int y; // another local variable
    }
}
    1. همچنین i هم یک متغیر local در نظر گرفته میشه و خارج از اسکوپ مربوط به for شناخته نمیشه.

    چرا لوکال یا گلوبال؟

    • در مواقعی که ما به طور موقت به یک متغیر احتیاج داریم (مثلا iدر forما اون رو به طور لوکال تعریف می کنیم تا بتونیم در آینده دوباره متغیری با همون نام ایجاد کنیم و به مشکل نخوریم.
    • اما وقتی که چند تابع از کدمون روی یک متغیر تاثیر میذارن بهتره اون رو گلوبال تعریف کنیم که کار راحت تر بشه.

    تذکر

    حواستون باشه که ما داخل یک اسکوپ نمیتونیم دو متعیر با یک نام تعریف کنیم. یعنی، 

    {
    int x = 2;
    int x = 3;
}
    این کد با خطای زیر مواجه میشه: 
    var.cpp:15:9: error: redefinition of 'x'
   15 |     int x = 2;
      |         ^
var.cpp:14:9: note: previous definition is here
   14 |     int x = 3;
      |         ^
1 error generated.

shell returned 1

    اما اگر متغیرهای با نام های یکسان رو در ۲ اسکوپ متفات تعریف کنید، کامپایلر مقدار اون رو برابر با متغیر تعریف شده در همون اسکوپ در نظر میگیره. یعنی:

    int x = 2;
cout << x; // output: 2
{
    int x = 3;
    cout << x; // output: 3
}

    اپراتور ها

    Arithmetic Operators

    Note

    عملگر توضیج
    a + b جمع
    a - b تفریق
    a * b ضرب
    a / b تقسیم
    a % b باقی مانده a بر b
    a++ افزایش تکی a
    b-- کاهش تکی b

    نکته

    همچنین در زبان ++C اولویت عملگر های ریاضی به همان صورت حفظ شده است. به عنوان مثال: 

    int a = 2 + 3 * 5;      // a: 17
int b = (2 + 3) * 5;    // b: 25

    همچنین اولویت % با * و / برابر است.

    Assignment Operators

    Note

    عملگر توضیج
    a = b مقدار a رو برابر با b قرار میده
    a += b مقدار a رو به اندازه b تا افزایش میده
    a -= b مقدار a رو به اندازه b تا کاهش میده
    a *= b مقدار a رو برابر با a * b قرار میده
    a %= b مقدار a رو برابر با a % b قرار میده
    a /= b مقدار a رو برابر با a / b قرار میده

    چند مثال:

    int a = 2, b = 3, c = 0;
c = a + b;  // c: 5
c += 2;     // c: 7 
a *= c;     // a: 14
b = a % b;  // b: 2
b++;        // b: 3
b /= 2;/*(1)!*/  // b: 1
    1. از آنجایی که b یک int یا عدد صحیح می باشد، طبق قرارداد، ++C در تقسیم مقدار رو برابر با جز صحیح اون تقسیم در نظر میگیره.

    Relational Operators

    این عملگر ها بین دو عدد قرار میگیرن و true یا false بر میگردونن، که به ما در جملات شرطی کمک زیادی میکنن.

    Note

    عملگر توضیج
    a == b بررسی تساوی
    a != b بررسی تفاوت
    a > b بررسی بزرگی اکید a نسبت به b
    a < b بررسی کوچکی اکید a نسبت به b
    a >= b بررسی بزرگتر یا مساوی بودن a نسبت به b
    a <= b بررسی کوچکتر یا مساوی بودن a نسبت به b

    با ترکیب این عملگر ها با (و)، (یا) و (NOT) منظقی که به ترتیب به (&&)، (||) و (!) نمایش داده می شوند می توان جملات شرطی نوشت.

    int x = 1, y = 2, z = 5;

bool f = ((x == 1) || (z + y <= 2)) && (y - 3 != 0);    // f: true
