# درسنامه

این درسنامه به عنوان منبع کمکی برای درس برنامه نویسی مقدماتی برای شما ارائه شده.

بعضی مباحث پیش نیاز مباحث دیگر هستن که در اون قسمت ها ما مبحث پیش نیاز رو در قالب [لینک]() قرار دادیم.

همچنین از سرچ بار :octicons-search-24: بالا میتونید برای پیدا کردن راحت تر مطالب استفاده کنید.

!!! info
    مباحث به ترتیب پیشنهادی در درسنامه قرار گرفته اما بهتره که شما ترتیب تدریس استاد خودتون از درسنامه استفاده کنید.
